Bonjour! Ce site est créé pour vous aider à convertir vos fichiers texte (epub, pdf, word) en fichiers mp3! Si vous préférez l'anglais ne vous inquiétez pas, une option en anglais est également disponible en cliquant sur "English" accessible sur toutes les pages.

Vous pouvez convertir vos fichiers en les téléchargeant sous l'onglet Télécharger votre fichier Téléchargez Votre Fichier. Le fichier peut soit être déposé dans le paramètre de la boîte de dépôt, soit en cliquant sur le bouton "Choisir un fichier" pour télécharger vos fichiers texte.

Si vous avez d'autres questions ou préoccupations concernant le site Web, n'hésitez pas à nous contacter en cliquant sur l'onglet Contactez-nous Contactez-Nous.
